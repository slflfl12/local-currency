# 경기도 지역화폐 MAP

## 프로젝트 내용

경기도 지역화폐 가맹점 지도 어플리케이션입니다.
주요 기능은 내 주변 3km 이내 까지 사용처 검색을 할 수 있습니다.
또한, 검색기능을 통해 가맹점을 검색할 수 있습니다.


## 기술 스택 & 라이브러리

* **Architecture**
  * **MVVM Architecture**
  * **Repository Pattern**
* **Libraries**
  * **JetPack**
    * **AAC ViewModel**
    * **AAC LiveData**
    * **AAC LifeCycle**
    * **Databinding**
    * **Room Persistence**
  * **Reactive Extension**
    * **RxJava**
    * **RxAndroid**
  * **ETC**
    * **Koin**
    * **Lottie**
    * **Gson**
* **Database**
    * **SQLite**
    * **SharedPreferences**
